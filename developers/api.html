توثيق API

1. المصادقة

للمصادقة على طلبات API، يجب تضمين مفتاح API في رأس الطلب:

Authorization: Bearer sk_live_51Np...

2. نقاط النهاية

أهم نقاط النهاية في API:

  • GET /api/v1/contacts - استرجاع جهات الاتصال
  • POST /api/v1/messages - إرسال رسالة
  • GET /api/v1/campaigns - استرجاع الحملات

3. أمثلة

مثال على إرسال رسالة باستخدام cURL:

curl -X POST https://api.wadi.com/v1/messages \
  -H "Authorization: Bearer sk_live_51Np..." \
  -H "Content-Type: application/json" \
  -d '{"to": "966501234567", "text": "مرحباً!"}'
اسم المستخدم user@example.com
